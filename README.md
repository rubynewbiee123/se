## 프로젝트 개요

### 배경
기존에 팀 내 파일 공유가 메신저, 이메일, USB 등으로 분산되어 있어 버전 혼동, 보안 취약, 접근성 부족 등의 문제가 발생하고 있었습니다. NextDriv는 이러한 문제를 해결하기 위해 자체 구축하는 파일 관리 시스템입니다.

### 목표
- 조직 내 200명 규모의 사용자가 동시에 안정적으로 사용할 수 있는 파일 스토리지 제공
- 파일 업로드/다운로드, 폴더 관리, 버전 관리, 권한 기반 공유 등 핵심 기능 구현
- 오픈소스 기술 스택 기반으로 라이선스 비용 없이 운영 가능한 구조 설계

### 주요 기능
- **파일 관리** — 업로드, 다운로드, 폴더 생성/이동/삭제, 드래그 앤 드롭 지원
- **버전 관리** — 파일 수정 이력 보관, 이전 버전 복원
- **공유 및 권한 제어** — 사용자별 열람/수정/댓글 권한 설정, 만료 기간이 있는 외부 공유 링크 제공
- **검색** — 파일명, 날짜, 업로더, 파일 유형 기준으로 빠른 검색 지원
- **사용자 계정 및 권한 관리** — 관리자의 계정 생성/삭제, 저장 공간 관리 및 그룹 기능
