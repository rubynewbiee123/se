# NextDriv 개발 프로젝트

> 조직 내 팀원들이 파일을 중앙에서 관리하고 공유할 수 있는 **클라우드 기반 파일 관리 시스템**

NextDriv는 약 200명 규모의 조직 내 구성원이 동시에 사용할 수 있는 웹 기반 클라우드 파일 관리 시스템입니다. 사용자는 웹 브라우저에서 다양한 형식의 파일을 업로드·다운로드하고, 폴더로 정리하며, 권한에 따라 팀원과 안전하게 공유할 수 있습니다.

## 🎯 프로젝트 개요

### 배경
기존 조직 내 파일 공유는 메신저, 이메일, USB 등으로 분산되어 다음과 같은 문제가 발생하고 있었습니다.

- **버전 혼선**: 같은 파일이 여러 사람의 컴퓨터에 분산되어 최신 버전 식별이 어려움
- **검색 비효율**: 파일이 분산 저장되어 필요한 파일을 찾는 데 많은 시간 소요
- **보안 취약성**: 이메일·메신저 공유 시 접근 권한 관리가 어렵고 외부 유출 위험 존재

NextDriv는 이러한 문제를 해결하기 위해 자체 구축하는 클라우드 기반 파일 관리 시스템입니다.

### 목표
- 조직 내 약 **200명 규모의 사용자**가 동시에 안정적으로 사용할 수 있는 파일 스토리지 제공
- 파일 업로드/다운로드, 폴더 관리, 버전 관리, 권한 기반 공유 등 **핵심 기능 8종** 구현
- **오픈소스 기술 스택**으로 라이선스 비용 없이 운영 가능한 구조 설계
- 보안성·안정성·성능을 균형 있게 고려한 안정적 협업 환경 구축

## 🛠️ 주요 기능

### 1. 파일 업로드/다운로드 (F01)
- 다양한 형식의 파일을 웹 브라우저에서 업로드·다운로드
- 단일 파일 최대 **2GB** 지원
- 대용량 파일을 **청크(Chunk) 단위로 분할 전송**하여 네트워크 불안정 시에도 안정적
- SHA-256 해시 기반 중복 파일 자동 탐지로 저장 공간 절약

### 2. 폴더 관리 (F02)
- 폴더 생성·이름 변경·삭제 지원
- 다단계(트리 구조) 폴더 구조 지원
- 드래그 앤 드롭으로 파일·폴더 이동

### 3. 파일 공유 및 권한 관리 (F03)
- 사용자별 **열람/수정/댓글** 권한 세분화 설정
- **외부 공유 링크 생성** — 만료 기간·비밀번호 설정 가능
- 권한이 없는 사용자의 접근은 자동 차단

### 4. 파일 검색 (F04)
- **파일명·날짜·업로더·파일 유형** 기준 다중 검색
- 1,000개 파일 기준 **1초 이내** 검색 결과 반환

### 5. 파일 버전 관리 (F05)
- 파일 수정 시 이전 버전 자동 보관
- 수정 이력 조회 및 **원하는 버전으로 복원** 가능
- 협업 과정에서 발생하는 실수 방지

### 6. 사용자 계정 및 권한 관리 (F06)
- 관리자에 의한 계정 생성·삭제
- 사용자별 저장 공간 할당량 설정
- 그룹 단위 권한 관리

### 7. 동시 접근 제어 (F07)
- **파일 잠금(Lock) 메커니즘**으로 동시 수정 충돌 방지
- 비활성 시 자동 잠금 해제 및 임시 버전 저장
- 충돌 발생 시 양쪽 버전 모두 보존하여 데이터 손실 방지

### 8. 휴지통 및 복구 (F08)
- 삭제된 파일을 **휴지통에 30일간 자동 보관**
- 사용자가 직접 휴지통에서 파일 복구 가능
- 30일 경과 후 자동 영구 삭제

## 🚀 빌드 및 실행 방법

> ⚠️ 현재 본 프로젝트는 **문서 작성 단계**에 있으며, 소스 코드 구현은 추후 진행 예정입니다. 아래는 향후 구현 시 적용 예정인 빌드·실행 절차입니다.

### 1. 저장소 클론
```bash
git clone https://github.com/rubynewbiee123/se.git
cd se
```

### 2. 환경 변수 설정
```bash
cd src
cp .env.example .env
```
> `.env` 파일을 열어 DB 접속 정보, JWT 시크릿 키 등을 입력합니다.

### 3. 의존성 설치
```bash
# Backend
cd src/backend
npm install

# Frontend
cd ../frontend
npm install
```

> 💡 상세 빌드 절차 및 배포 가이드는 `src/` 디렉토리 추가 후 `src/README.md`에 별도로 작성됩니다.
